# CLI

This project uses [Typer](https://typer.tiangolo.com/) and [Click](https://click.palletsprojects.com/) for CLI functionality. When the project is installed the cli is available at `ali_backend`.

The full help contents can be visited with the help flag.

```bash
ali_backend --help
```

The CLI itself is defined at `ali_backend.cli`. New commands can be added there.
